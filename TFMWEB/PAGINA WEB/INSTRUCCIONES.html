Instrucciones juego POR/PARA

Y ahora ¡vamos a aprender cómo se juega! Es muy fácil, pero como en todos los juegos, hay que seguir algunas reglas.

Para jugar a este juego necesitas:

  1. Utilizar el ratón para girar. También puedes usar el panel táctil

  2. Utilizar las flechas flechas del teclado para moverte hacia adelante, hacia atrás y a los lados.

  3. Para iniciar una conversación debes "tocar" a los personajes/objetos con los que se pueda interactuar (Capitana Leandra, diagramas e imágenes) y pulsar la barra espaciadora.

  4. Pulsar la tecla de mayúsculas para continuar una conversación.
